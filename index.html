04 — Signal Processing
Preprocessing Pipeline
Raw EEG is noisy. These steps clean it before any analysis.
1
Bandpass Filter (0.5 – 40 Hz)
Removes DC drift below 0.5 Hz and high-frequency muscle/EMG noise above 40 Hz. Use a 4th-order Butterworth filter. Implemented with scipy.signal.butter() in Python.
2
Notch Filter (50 Hz or 60 Hz)
Suppresses power line interference (50 Hz in Bangladesh/South Asia). Apply a narrow notch (Q=30) at the mains frequency using scipy.signal.iirnotch().
3
Artifact Removal
Remove eye blink artifacts (large amplitude spikes in the frontal channel) using amplitude thresholding or Independent Component Analysis (ICA). Use MNE-Python or EEGLAB for automated ICA.
4
Epoching / Segmentation
Cut continuous signal into fixed-length windows (e.g., 1-second epochs with 50% overlap). Each epoch becomes one sample for ML. Common: 256 samples @ 256 Hz = 1 second.
5
Feature Extraction
Compute band power via Welch's PSD, Hjorth parameters (activity, mobility, complexity), statistical features (mean, variance, skewness), or wavelet coefficients for time-frequency features.

08 — Implementation
Sample Code Snippets
Python — EEG Bandpass + Band Power Extraction 
import numpy as np
from scipy.signal import butter, filtfilt, welch

def bandpass_filter(data, lowcut, highcut, fs, order=4):
    """Apply Butterworth bandpass filter to EEG data."""
    nyq = 0.5 * fs
    b, a = butter(order, [lowcut / nyq, highcut / nyq], btype='band')
    return filtfilt(b, a, data)

def extract_band_powers(epoch, fs=250):
    """Compute power in each EEG frequency band (Welch method)."""
    freqs, psd = welch(epoch, fs=fs, nperseg=fs)
    bands = {
        'delta': (0.5, 4), 'theta': (4, 8),
        'alpha': (8, 13), 'beta': (13, 30), 'gamma': (30, 45)
    }
    powers = {}
    for band, (lo, hi) in bands.items():
        idx = np.logical_and(freqs >= lo, freqs <= hi)
        powers[band] = np.trapz(psd[idx], freqs[idx])  # area under PSD
    return powers

# Example usage
raw_eeg = np.random.randn(250)   # 1 second at 250 Hz
filtered = bandpass_filter(raw_eeg, 0.5, 40, fs=250)
features = extract_band_powers(filtered)
print(features)
# {'delta': 0.42, 'theta': 0.18, 'alpha': 0.35, 'beta': 0.21, 'gamma': 0.08}

import network, time, json
from umqtt.simple import MQTTClient

SSID = "YourWiFi"
PASSWORD = "YourPass"
MQTT_BROKER = "192.168.1.100"
TOPIC = b"eeg/raw"

def connect_wifi():
    wlan = network.WLAN(network.STA_IF)
    wlan.active(True)
    wlan.connect(SSID, PASSWORD)
    while not wlan.isconnected():
        time.sleep(0.5)

connect_wifi()
client = MQTTClient("eeg_node", MQTT_BROKER)

# X: shape (n_epochs, n_features) — e.g., 5 band powers per epoch

09 — Getting Started
Your Project Roadmap
A practical step-by-step path from zero to a working EEG-IoT system.
1
🛒 Buy a NeuroSky MindWave (or use OpenBCI if budget allows)
Start with a NeuroSky. Install the ThinkGear connector SDK. Verify raw EEG streaming works to your PC via Bluetooth before adding any IoT layer.
2
🐍 Set up Python environment with MNE + BrainFlow
Install: pip install mne brainflow scipy scikit-learn matplotlib. Use BrainFlow's Python API to connect to your headset and stream live data to a CSV file.
3
⚙️ Build the signal processing pipeline
Apply bandpass (0.5–40 Hz), notch (50 Hz), and epoch the signal into 1-second windows. Extract the 5 band power features per epoch. Save as a feature matrix.
4
🏷️ Collect labeled data for your task
Record 5–10 minutes of EEG per class (e.g., eyes open = 0, eyes closed = 1). Label each epoch. Aim for at least 200 epochs per class. Use balanced classes.
5
🤖 Train and evaluate an SVM or Random Forest
Start simple with scikit-learn SVM. Achieve baseline accuracy. Then experiment with 1D-CNN if performance is insufficient. Use 5-fold cross-validation.
6
📡 Add IoT layer with ESP32 + MQTT
Write a MicroPython script on the ESP32 that receives serial EEG data and publishes to an MQTT broker. Subscribe from a Flask server on your PC or Raspberry Pi.
7
📊 Build a live dashboard or trigger IoT actions
Use Flask + Chart.js to display real-time band power bars, or connect the classifier output to GPIO pins that control relays, LEDs, or home automation systems.
